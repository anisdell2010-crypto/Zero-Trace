## 🌟 Key Features
- **Identity Spoofing:** One-click MAC randomization and monitor mode activation.
- **Dynamic VPN Check:** Real-time detection of secure tunnels (Tun0/Proton/WireGuard) to prevent IP leaks.
- **Forensic Sanitization:** Automated purging of RAM caches, system logs, and history files.
- **Emergency Self-Destruct:** Multi-pass `shred` operation to eliminate the tool and its traces instantly.
